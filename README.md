# HomeLibrary

HomeLibrary is a Java desktop application for managing a personal home library using MySQL.

## Features

- Add new books
- Edit existing books
- Delete books
- Search books by title, author, genre or ISBN
- View all books
- Store information such as:
  - Title
  - Author
  - Genre
  - Publisher
  - Publication year
  - ISBN
  - Number of pages
  - Reading status
  - Personal rating
  - Notes
  - Review

## Technologies

- Java
- MySQL
- JDBC
- Eclipse IDE
- Git
- GitHub
